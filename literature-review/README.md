# Literature Review


This folder contains materials used in the literature review of QST. The review includes **62 QST papers** and examines how existing testing techniques address
the challenges and practical needs identified through our interviews and forum analysis.


## Directory Structure

```text
Literature-Review/
├── selected_papers.xlsx
├── keywords.pdf
└── README.md
```

## File Descriptions

### `selected_papers.xlsx`

This file contains the bibliographic information (e.g., paper title, publication year, publication venue) of the **62 papers** included in the literature review.


### `keywords.pdf`

This file lists the **30 search keywords** used in the keyword-based search for 
the literature review. 
The keywords were developed through a preliminary examination of 
QST studies.
